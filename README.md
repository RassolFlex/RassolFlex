Я - Python-разработчик. Более 2,5 лет занимаюсь разработкой на Python, из них более года - коммерческой разработкой.

Специализируюсь на backend разработке на Python/Django, работе с PostgreSQL, интеграции внешних API и сопровождении веб-проектов полного цикла - от реализации бизнес-логики до развёртывания на Linux серверах.

Имею опыт поддержки и развития коммерческих веб-проектов, настройки Nginx, Supervisor, DNS-инфраструктуры и работы с Git/GitLab. Разрабатывал внутренние API, интеграции с Telegram, ВКонтакте и 1С ERP, автоматизировал обработку больших объёмов данных и файлов.

Уверенно работаю в Linux-среде и быстро осваиваю новые технологии. В разработке использую современные инструменты автоматизации и ИИ для повышения эффективности работы и качества кода.
